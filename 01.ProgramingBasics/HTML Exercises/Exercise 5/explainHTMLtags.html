HTML commands:

< img >
  • purpose: embeds an image into the document
  • example: < img src="../fluffy_cat.jpg" height="20%" alt="sweet fluffy cat " >
  • attributes:
Attribute Value Meaning
src URL source file
height number, percentage The intrinsic height of an image, in pixels. Must be an integer without a unit.
width number, percentage The intrinsic width of an image, in pixels. Must be an integer without a unit.
alt text Defines an alternative text description of an image.
< a > , < /a >
  • purpose: anchor element, creates a hyperlink to webpages, files, email addresses, locations in the same page, or anything else a URL can address.
  • example: < a href= " https://en.wikipedia.org/wiki/Cat " > Wiki Cats < / a >
  • attributes:
>
Attribute Values Meaning
href URL The URL that the hyperlink points to.
target name_of_frame Where to display the linked URL, as the name for a browsing context (a tab, window, or < frame > )
_blank usally a new tab, but users can configure browsers to open in a new window instead
_parent the parent browsing context of the current one. If no parent, behaves as _self.
_top the topmost browsing context (the " highest " context that is an ancestor of the current one). If no ancestors, behaves as _self.
_self the current browsing context. (Default)
